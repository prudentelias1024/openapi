Here are the lists and overview of all the HTTP methods our API accepts

Get all products

fetch('https://openapi.com/products') .then(res=>res.json( ) ) .then(json=>console.log(json) )

Get a single product

fetch('https://openapi.com/products/1') .then(res=>res.json( ) ) .then(json=>console.log(json) )

Get certain number of products

fetch('https://openapi.com/products/max=5') .then(res=>res.json( ) ) .then(json=>console.log(json) )

Add a product

fetch('https://openapi.com/products/1', { method: "POST", body: JSON.stringify({ id: 21, name: "Nike Boots", size: "S", catchword: "Nice boots!", productOwner: "Finn Blake", dateUploaded: "Sept 13, 2003" }) }) .then(res=>res.json( ) ) .then(json=>console.log(json) )

Change a product's price

fetch('https://openapi.com/products/1/price', { method: "PUT", body: JSON.stringify({ price: "$43.21", }) }) .then(res=>res.json( ) ) .then(json=>console.log(json) )

Update a product

fetch('https://openapi.com/products/1', { method: "PUT", body: JSON.stringify({ id: 21, name: "Nike Boots", size: "S", catchword: "Nice boots!", productOwner: "Finn Blake", dateUploaded: "Sept 13, 2003" }) }) .then(res=>res.json( ) ) .then(json=>console.log(json) )

Delete a product

fetch('https://openapi.com/products/1', { method: "DELETE", body: " " }) .then(res=>res.json( ) ) .then(json=>console.log(json) )

Delete a Todo task
